# auto-generator-tips

### 1.概要  
Auto Generatorのツールをシェルスクリプトで作ったときに利用したTips

### 2.紹介

#### 2-1.parts-writer.sh  
特定のファイルの情報を別ファイルの特定行に差し込みたい時に利用

#### 2-2.delete-line.sh  
特定のファイルの内容を複数行まとめて削除したい時に利用

#### 2-3.property-loader.sh  
propertyファイルの中身を読み込む際に利用

#### 2-4.property-register.sh  
propertyファイルの中身に書き込む際に利用
